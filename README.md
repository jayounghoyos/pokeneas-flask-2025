# pokeneas-flask-2025


## How to Run

It's simple, but we wrote this for good practice.

- Clone the repo:
```bash
git clone https://github.com/jayounghoyos/pokeneas-flask-2025.git
```
- Copy the env example:
```bash
cp envExample .env
```
- Install requirements:
```bash
pip install -r requirements.txt
```
- Start the app:
```bash
python run.py
```
